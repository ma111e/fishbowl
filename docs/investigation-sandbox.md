# Investigation Sandbox

The sandbox is a visual workspace for linking entities across pages and
sessions. Drag detected entities onto a canvas and organize the relationships
between them.

## Open and populate

From [execution mode](keyboard-shortcuts.md) (`Ctrl+X`):

| Key | Action |
|---|---|
| `O` | Open the sandbox |
| `N` | Create a new investigation from the current page's entities |
| `I` | Import the current page's entities into the active investigation |

## Working in the canvas

- Drag entities onto the graph and arrange them.
- View each entity's enrichment blocks inline.
- Link related indicators to map an investigation.
- Correlate entities seen across different pages.

## Layouts

Switch between layout engines to arrange the graph:

- **Flow**
- **Force-directed**
- **Clustered grid**

## Import / export

Investigations export to and import from JSON, so you can save, share, or reload
a workspace.
